# Screenshots Guide

This directory is for storing screenshots of the application. To generate screenshots:

## Web App Screenshots

1. Start the web app: `npm run dev:web`
2. Open http://localhost:5173
3. Take screenshots of:
   - Dashboard with issues list
   - Issue detail view
   - Analytics widget
   - Filter/search functionality

## Mobile App Screenshots

1. Start the mobile app: `npm run dev:mobile`
2. Run on iOS Simulator or Android Emulator
3. Take screenshots of:
   - Report screen (empty form)
   - Report screen with filled data and images
   - Success screen
   - Offline queue notification
