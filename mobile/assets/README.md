# Mobile App Assets

This directory should contain the following assets for the Expo app:

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen (1242x2436 recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

You can generate these using:
- [Expo Asset Generator](https://www.npmjs.com/package/@expo/asset-generator)
- Or create them manually using design tools

For now, Expo will use default assets if these are missing.
